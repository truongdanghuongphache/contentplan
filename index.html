Dùng Supabase để lưu trên cloud — miễn phí, không cần thẻ.
Chỉ setup 1 lần, sau đó tự động sync khi mở app trên bất kỳ thiết bị nào.

Cách lấy thông tin Supabase (5 phút):
1. Vào supabase.com → Đăng nhập / tạo tài khoản
2. Nhấn New Project → đặt tên → chọn region → Create
3. Vào Project Settings → API
4. Copy Project URL và anon/public key dán vào dưới
5. Vào SQL Editor → chạy câu lệnh tạo bảng bên dưới
CREATE TABLE IF NOT EXISTS content_planner ( id TEXT PRIMARY KEY, data JSONB NOT NULL, updated_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE content_planner ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_access" ON content_planner FOR ALL USING (true);
